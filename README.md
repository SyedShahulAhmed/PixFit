# PixFit

PixFit is a simple tool for creators to optimize images and videos for social media platforms.

### Features

* 🖼️ Resize and format images for social media
* 🎥 Compress videos to reduce file size
* 📱 Platform-specific presets (Instagram, Facebook, X, YouTube, and more)
* ⚡ Fast and easy-to-use interface
* ☁️ Cloud-based media processing


### Goal

Create social-media-ready content in seconds with the right dimensions and optimized file sizes.
